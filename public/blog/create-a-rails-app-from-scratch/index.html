Create a Rails App From Scratch

Nov 26 2021
  1. Create a folder where you want the rails app to live.
> mkdir rails-app
> cd rails-app
  1. Run this code in your terminal to create your app. Specify the databsae you want (I always use PostgreSQL) and I also specify –edge so I use the most recent version of Rails.
> rails new rails-app --database=postgresql --edge
  1. Then set up your database and migrate it.
> bin/rails db:migrate
> bin/rails db:setup
  1. To run your Rails server, run this. You should see an image that says “Yay, you’re on Rails!”
> bin/rails server
