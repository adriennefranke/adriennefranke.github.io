Have you ever messed up your local Django SQLite database so badly that you just need to toss it? ðŸ—‘

  1. Delete db.sqlite3 in your Django project folder.

  2. Delete everything in the migration folder in each Django app. However, do not remove the __init__.py files.

  3. Make any changes to your models in models.py.

  4. Make the migrations.

     > python manage.py makemigrations

  5. Migrate.

     > python manage.py migrate

  6. Create a new superuser.

     > python manage.py createsuperuser
